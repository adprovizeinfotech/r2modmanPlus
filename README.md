## 🧩 What Are Mods?

Mods are user-created changes or additions that can improve or change how games work.

Examples of mods include:

- Adding new items or characters  
- Changing game rules or difficulty  
- Fixing bugs or improving performance  
- Adding new missions or maps  

Using r2modmanPlus helps you find trusted mods and manage them safely.
